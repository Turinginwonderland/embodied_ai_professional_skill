# /report 命令

调用 `synthesis-output-generation` skill，按 `$ARGUMENTS` 主题撰写行业深度报告。

## 执行步骤

1. **思路梳理**：从 `kb_*.md` 抽取与主题相关的条目，构建"技术背景→核心痛点→解决方案→市场空间→代表企业"逻辑链
2. **MECE 大纲**：构建相互独立完全穷尽的章节结构，每个大纲节点批注核心事实与数据支持（关联条目 ID）
3. **章节确认**：复杂报告分段生成，每完成一节向用户确认
4. **报告撰写**：行业分析师/战略专家笔调，语言平实严谨，用数据与事实说话

## 约束

- 严禁使用未在知识库中登记或证实的虚假数据
- 术语需符合具身智能行业规范（VLA / World Model / Sim-to-Real / 微操控制 / 泛化能力等）
- "已废弃"条目不引用；"冲突-待裁决"条目需双轨呈现
