# Skill: Synthesis & Output Generation (综合与研究输出)

## 1. 技能定位
本技能负责调用已被更新的知识库，梳理研究思路，构建研究大纲，并最终输出符合行业标准的深度研究报告。

## 2. 核心工作流
1. **思路梳理（思维链构建）**：
   - 根据研究主题（如：人形机器人手部关节的技术路线演进），梳理从“技术背景 -> 核心痛点 -> 解决方案 -> 市场空间 -> 代表企业”的逻辑链。
2. **大纲构建（MECE原则）**：
   - 产出结构化的大纲，确保各章节“相互独立，完全穷尽”（MECE）。
   - 每个大纲节点下必须批注：此处应调用的核心事实与数据支持。
3. **报告撰写（专业化成文）**：
   - 采用行业内分析师或战略专家的专业笔调，语言平实、严谨，多用数据与事实说话。
   - 避免空洞的套话，确保技术名词使用准确（如：谐波减速器、灵巧手、空心杯电机等）。

## 3. 约束条件
- 严禁在大纲或报告中使用未在“输入板块”中登记或证实的虚假数据。
- 复杂报告必须分段生成，每完成一个章节需提示用户确认，避免篇幅过长导致质量下降。s
