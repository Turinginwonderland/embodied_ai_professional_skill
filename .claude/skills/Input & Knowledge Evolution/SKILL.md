# Skill: Input & Knowledge Evolution (输入与知识演进)

## 1. 技能定位
本技能负责接收新的具身智能相关素材（PPT、研报、新闻、政策），提取事实，并与现有知识库进行比对，完成观点的迭代与“破旧立新”。

## 2. 核心工作流
1. **结构化提取**：
   - 提取新输入材料中的核心论点、技术路线（如：从端到端转向分层控制）、关键数据（如：量产成本、传感器方案）。
2. **新旧对比分析（比对与纠偏）**：
   - 将新提取的观点与已有研究结论进行交叉验证。
   - 识别出三类信息：
     - **证实（Confirm）**：增强了已有观点的证据。
     - **互补（Complement）**：填补了此前研究的空白。
     - **冲突/纠偏（Conflict）**：新观点/新数据推翻了旧观点（例如：最新官方政策调整了标准，或某厂商更新了技术路径）。
3. **知识库更新更新机制（破旧立新）**：
   - 当出现冲突时，遵循“最新、最权威优先”原则，明确标记“旧观点已被废弃/修正，新观点为：...，理由是：...”。

## 3. 约束条件
- 必须注明新观点的来源出处（如：某券商202X年X月研报，或某官方发布会）。
- 在没有充足客观证据支持前，不要轻易将猜测性的新观点作为定论，应标记为“待观察趋势”。
